# Observation Options and Short Codes

> Source: Configuration Document for SPOT NPEWS (NHS England)

---

## Options for Skipping an Observation

### Blood Pressure

1. **Not Attempted (No Concern)** (NCO) – Scores 0
2. **Unsuccessful Attempt (No Concern)** (U0) – Scores 0
3. **Unsuccessful attempt (Concern)** (U4) – Scores 4
4. **Other** (Text Box to complete) – user selects whether to score 0 or 4

### Carer Question

1. Skipped (no reasons presented)

### Clinical Intuition

1. Skipped (no reasons presented)
