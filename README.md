# digital-pews

An RCPCH Incubator project to develop an open source implementation of the National PEWS (Paediatric Early Warning Score) system, incorporating SPOT (System for Patient Observation and Tracking) principles.

## Methodology for implementation

Source documents are stored in this repository, in the [reference-sources](./reference-sources) directory. These are the basis for the implementation of the digital PEWS system, and were used to create the specification documents in the [spec](/spec) directory.

* Source documents were read and analysed to understand the general shape of the requirements, and to identify the key components of the system.
* Human-guided LLM agents were used to extract and format the specification into clear Markdown documents, without changing any of the wording or meaning. A small number of grammatical and spelling errors were corrected, but the content was not changed in any way.
* These specification documents were then used to develop a comprehensive and detailed [roadmap.md](./roadmap.md) for the implementation of the digital PEWS system.
